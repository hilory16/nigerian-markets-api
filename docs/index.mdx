# Iya Oloja API

Iya Oloja is a free JSON API for discovering markets across Nigeria. You can browse states, drill into LGAs, list markets, search by name, and submit new market suggestions for review.

## Why this API exists

- Public market data in Nigeria is fragmented and often hard to query programmatically.
- This API gives you a simple, unauthenticated way to explore community-maintained market records.
- The source data stays open in the repository, so every change is inspectable and versioned.

## Base URL

```bash
https://iya-oloja.pages.dev/api
```

## What you can do

- List all states and the FCT
- Fetch a state together with its LGAs
- Fetch an LGA together with its parent state and markets
- Page through all markets
- Search by market, LGA, or state name
- Submit a market suggestion through the contribution endpoint
